Skills




HTML (HyperText Markup Language) is the foundation of web development, used to structure and display content on web pages. It defines the layout and elements of a webpage, working alongside CSS (for styling) and JavaScript (for interactivity).

CSS (Cascading Style Sheets) is a styling language used to design and layout web pages. It controls the appearance, colors, fonts, spacing, animations, and responsiveness of HTML elements.

JavaScript (JS) is a powerful, high-level programming language primarily used for web development. It enables dynamic and interactive features on websites, working alongside HTML and CSS. JavaScript runs in the browser but can also be used on the backend with Node.js.

React is a JavaScript library for building fast and interactive user interfaces (UIs). Developed by Facebook (Meta), it follows a component-based architecture that makes it easy to build reusable UI elements.

Python is widely used in web development due to its simplicity, flexibility, and vast ecosystem of frameworks. It is primarily used for backend development, handling server-side logic, databases, and API integrations.

Canva is a popular graphic design platform that allows users to create professional-quality designs with ease. It offers a drag-and-drop interface, pre-made templates, and a vast library of design elements like images, icons, fonts, and animations.
